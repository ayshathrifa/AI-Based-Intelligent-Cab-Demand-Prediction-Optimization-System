About CabDemand AI

Machine learning powered cab demand forecasting system

🎯 Project Overview

CabDemand AI is an intelligent cab demand forecasting system that uses machine learning to predict ride demand across city zones. It helps cab operators optimize driver allocation, reduce wait times, and maximize revenue by predicting demand before it happens.

🛠️ Technology Stack

Python / Flask Scikit-learn Random Forest Linear Regression HTML / CSS / JS Chart.js SQLite REST API
🤖

ML Models

Random Forest & Linear Regression trained on real cab demand data

📊

Data-Driven

Predictions based on time, location, weather, and historical patterns

⚡

Real-Time

Instant predictions with auto-detected time and location parameters

📋 Features Summary

  • ✅ Demand prediction with ML
  • ✅ Real-time auto prediction
  • ✅ Zone-based demand analysis
  • ✅ Interactive demand heatmap
  • ✅ Driver allocation suggestions
  • ✅ Peak hour analysis
  • ✅ AI-powered insights
  • ✅ Data visualization charts
  • ✅ Admin panel with model training
  • ✅ Model comparison (RF vs LR)
